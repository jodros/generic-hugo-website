A Grammar for Any Language



The case for such grammar

So you want to learn new languages or maybe even teach them?

I’ve seeking for some way to learn new stuff on my own a couple of times throughout my life, but up to some months ago it was being hard somehow. I had found “An Introduction to Descriptive Linguistics” by H. A. Gleason in an old-book store and begun to be more interested in the subject, particularly the concept of a abstract and formal reduction of language (and complex structures in general as well) to the pure and simple description of each of its parts.

I mean, learning first how all natural languages works in its more deep gears, and then speed up the learning of any of them: inversely of what some pleople think, natural languages are distinguished by differences in its syntaxes and not in the lexical body. So learn foreign language is firstly about get how the sense is build according to its inner rules. The vocabulary and other “external” features comes just after that.

Basic questions:

  1. Is it a synthetic language or an analytic one?

Synthetic languages best-know examples are german, latin, greek and russian: the syntatic function of the words are defined by suffixation and word order does not have great importance. Whereas in analytic languages such as portuguese, italian and mandarim the position of the word has a fundamental role for its right meaning. It may look a bit hard at the beginning for someone used only to the analitic ones get who is who in a pure synthetic sentence lacking of any order.

  1. Has it some relation to your mother tongue?

Obviously, it’s much easier for a russian native speaker to learn ukrainian than for a spanish native speaker, but on the other side, for the last it’s much easier to learn portuguese than for the first…

Which features underlies every sentence in any language?

  1. An agent
  2. An action
  3. The object of such action

From where to begin?

The first step is, of course, learn the invariable words and the most common ones, a great way to get such set of words is consulting a Swadesh list, you can even generate multiple lists, side-by-side, for the languages you are searching on using python Natural Language Toolkit . Here are some examples of what you can do with it:

  • Romance languages
  • Slavic languages
  • Germanic languages

The base script for generate such pdf is the follow, you just need to change the ISO 639-1 language codes:

see the code 

  from nltk.corpus import swadesh 
  from tabulate import tabulate

  table = {"latin": swadesh.words("la"),
           "portuguese": swadesh.words("pt"),
           "spanish": swadesh.words("es"),
           "french": swadesh.words("fr"),
           "italian": swadesh.words("it"),
           "romanian": swadesh.words("ro")}
           
  tb = tabulate(table, headers='keys', tablefmt='grid')

  with open("swadesh.md", "w") as sl:
      sl.write(r'\pagenumbering{gobble}')
      sl.write("\n\n")
      sl.write(tb)
      sl.close()

  # pandoc file.md -o file.pdf
